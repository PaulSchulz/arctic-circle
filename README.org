* arctic-circle

Implementation of the Arctic Circle / Aztek Diamond graph interation 

[[images/arctic-circle.png]]

** Building
#+begin_src
cd src && make
#+end_src

** Running
#+begin_src 
./src/arctic-circle
#+end_src

Size and number of steps to run can be varied in the code by editing
#+begin_src
#define MAXX 81
#define MAXY 81
#define STEPMAX 40
#+end_src
